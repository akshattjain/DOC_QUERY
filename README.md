# DOC_QUERY
AI-powered document query system with a modern frontend and FastAPI backend, enabling intelligent document understanding, semantic search, and LLM-based responses.

---

##  Features

-  Upload and process documents (PDF, text, etc.)
-  Semantic search over document content
-  LLM-powered question answering
-  FastAPI backend for high-performance APIs
-  Modern frontend UI for seamless interaction
-  End-to-end full-stack integration

---

##  Tech Stack

### 🔹 Frontend
- React / Vite
- Tailwind CSS
- Axios

### 🔹 Backend
- FastAPI
- Uvicorn
- LangChain / LLM Integration
- Python

### 🔹 AI/ML
- Embeddings-based retrieval
- Vector search
- Retrieval-Augmented Generation (RAG)
